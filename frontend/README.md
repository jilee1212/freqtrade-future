# Freqtrade Future - Modern Frontend

Next.js 14 기반의 현대적인 트레이딩 대시보드

## 🚀 빠른 시작

### 개발 환경 실행
```bash
npm install
npm run dev
# http://localhost:3000
```

## 🎨 기술 스택
- Next.js 14 + TypeScript
- shadcn/ui + Tailwind CSS
- Freqtrade API 연동

## 📁 주요 페이지
- `/` - 홈페이지
- `/dashboard` - 트레이딩 대시보드

## 🔌 환경 변수
```env
NEXT_PUBLIC_FREQTRADE_URL=http://141.164.42.93:8080
```
